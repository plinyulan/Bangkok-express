### Project overview

Bangkok Express is a parcel delivery system operating within Bangkok, supporting parcel submission, pickup services, cost calculation, payment processing, and delivery tracking.The system provides end-to-end visibility across pickup, sorting, and delivery stages to ensure reliable and transparent logistics operations.

### My Role
Responsible for UI design and frontend development of a web-based editor interface.
Designed screens and interaction flows for managing parcel data and delivery statuses, and implemented structured frontend components aligned with system workflows.
